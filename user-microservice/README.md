USER-MICROSERVICE 
=================

Quick start
1. `mvn clean install`
2. `cd..`
3. `cd  docker`
3. `docker-compose up -d user`

Version 0.0.1
-------------
Technical info
--------------
* Database engine: Mysql
* Database schema : users
* Service port: 8881

Requirements
------------
* [Java Platform (JDK) 8](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
* [Apache Maven 3.x](http://maven.apache.org/)
